# Project Setup

Use when creating a new WinUI 3 C# app.

- Create project via Visual Studio template: WinUI 3 Blank App, Packaged.
- Prefer latest Windows App SDK unless project constraints.
- Decide packaging early (packaged vs unpackaged) based on deployment needs.

Source pointers (read when needed):
- https://learn.microsoft.com/en-us/windows/apps/tutorials/winui-notes/
- https://learn.microsoft.com/en-us/windows/apps/windows-app-sdk/support
