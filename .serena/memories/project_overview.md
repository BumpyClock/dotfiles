# Project Overview
- Purpose: personal dotfiles repository for configuring shells, tools, and agent/mcp setup across platforms.
- Tech stack: shell scripts (bash/zsh), PowerShell scripts, Python utility scripts, markdown docs.
- Structure highlights: .github/ (prompts/workflows), .claude/ (Claude config), mcpServers/ (MCP configs + sync), agents/ (agent defs), docs/ (language/source control docs), zshrc.d/ (zsh config).
- Platform: Linux (per onboarding context), but repo contains Windows PowerShell support.
