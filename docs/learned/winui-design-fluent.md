# WinUI design and Fluent guidance (2026-02-18)

## Sources
- https://learn.microsoft.com/en-us/windows/apps/design/
- https://learn.microsoft.com/en-us/windows/apps/design/design-principles
- https://learn.microsoft.com/en-us/windows/apps/winui/

## Findings
- Design hub frames guidance around user expectations and Fluent Design, with guidance for layout, navigation, input, typography, and motion, and emphasizes consistent interfaces across devices, input types, and form factors.
- Windows 11 design principles: Effortless, Calm, Personal, Familiar, Complete + coherent.
- Signature experiences: color, elevation/layering, iconography, materials, geometry, typography, motion.
- WinUI is a modern native UI framework delivering Fluent Design, high-performance rendering, and XAML to C# and C++ developers, and targets Windows 10 version 1809+ including Windows 11.
